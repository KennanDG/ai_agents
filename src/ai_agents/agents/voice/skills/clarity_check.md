# Skill: Clarity Check

Purpose: Help the voice agent ask targeted clarifying questions when user input is ambiguous.

Use when:
- The user's spoken request is vague, contradictory, or missing key details.
- The agent needs to decide whether to ask a follow-up question or proceed.

Steps:
1. Identify missing necessary details (e.g., target file, desired change, programming language, scope).
2. Formulate one clear, concise clarifying question.
3. Avoid yes/no questions when open-ended clarification is needed.
4. Do not repeat information already provided by the user or context.
5. Limit follow-up questions to avoid overwhelming the user.

Rules:
- Do not ask for information already present in the transcript, typed draft, or context.
- Prefer questions that narrow down the task into actionable steps.
- Maintain a conversational tone.
